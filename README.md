# SilentReed MoviePilot 插件仓库

MoviePilot 第三方插件库，提供通知和签到相关功能扩展。

## 插件列表

| 插件 | 版本 | 说明 |
|------|------|------|
| Server酱³通知 | 1.8.1 | 通过 Server酱³ 发送消息通知 |
| 绿联论坛积分助手 | 2.0.0 | 每日定时查询绿联论坛积分变化 |

## 安装方式

1. 打开 MoviePilot 设置 → 插件 → 插件源
2. 添加插件源：`https://github.com/SilentReed/MoviePilot-Plugins`
3. 在插件市场搜索插件名称并安装

## 插件一：Server酱³通知 (serverchan)

通过 Server酱³ 发送消息通知，支持 APP 推送。

### 功能

- 📥 下载任务添加通知
- 🗑️ 下载任务删除通知
- ✅ 媒体整理完成通知
- 📺 订阅完成通知
- ❌ 系统错误通知
- 💬 用户消息通知

### 配置

| 参数 | 说明 | 获取方式 |
|------|------|----------|
| UID | Server酱³ 用户 ID | https://sc3.ft07.com/ 注册获取 |
| SendKey | Server酱³ SendKey | 同上 |
| 消息类型 | 选择需要接收的通知类型 | 不选则接收所有 |

## 插件二：绿联论坛积分助手 (ugnascredits)

每日定时查询绿联论坛 (club.ugnas.com) 积分变化并推送通知。

### 功能

- 🕐 定时查询积分（支持自定义 Cron 表达式）
- 📊 积分变化追踪（记录历史积分数据）
- 📈 趋势分析（连续增长天数、上涨/下跌统计）
- 📱 消息推送通知
- 🔄 自动重试机制
- 📋 积分历史详情页
- 🔐 OAuth API 自动登录（无需手动抓 Cookie）
- 🌐 Playwright 浏览器兜底登录
- 👤 用户资料展示（头像、用户组、主题数、回帖数、好友数）

### 配置

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 启用插件 | 开启/关闭插件 | 关闭 |
| 发送通知 | 是否推送积分变化通知 | 开启 |
| 立即运行一次 | 测试运行一次后自动关闭 | 关闭 |
| 定时规则 | Cron 表达式 | `0 8 * * *`（每天08:00） |
| 用户名 | 绿联论坛用户名/手机号 | - |
| 密码 | 绿联论坛密码 | - |
| 论坛Cookie | 从浏览器获取的登录 Cookie（可选） | - |
| 用户UID | 绿联论坛用户 UID（可选，自动获取） | - |
| 历史保留天数 | 保留多少天的历史记录 | 30 |

### 说明

绿联论坛无需签到，登录即有积分。本插件仅查询积分变化并记录。
